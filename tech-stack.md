
# Tech Stack

## Frontend
- **React** with **TypeScript**
- **Tailwind CSS** — utility-first styling

## Backend
- **Node.js** with **Express** and **TypeScript**
- **DB Sessions** — session-based authentication stored in PostgreSQL

## Database
- **PostgreSQL** — relational database
- **Prisma** — ORM for type-safe database access

## AI
- **Claude API** (Anthropic) — ticket classification, summarization, and suggested replies

## Email
- **SendGrid** or **Resend** — for receiving support emails and sending responses

## Deployment
- **Docker** — containerized deployment for both frontend and backend
